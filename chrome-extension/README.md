# YourNotes Quick Portal

Chrome Manifest V3 companion for fast browser capture.

## Install locally

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Choose **Load unpacked** and select this `chrome-extension` folder.

Quick captures stay in Chrome's local extension storage. The context menu can save selected text, pages, and links. Features that need the complete data model open the latest full-app release.
